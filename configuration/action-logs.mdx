# Action Logs

Action logs allow you to monitor activities on your server by logging specific events to a designated channel. This helps you keep track of moderation actions, message updates, member changes, and more.

## Setup

1. **Enable the module** in the dashboard to start logging.

<img height="200" src="https://i.imgur.com/wfX14ZY.png" />

2. **Select a channel** where the log messages will be sent.

<img height="200" src="https://i.imgur.com/c9HAYyh.png" />

3. **Select the events** you want to track. Each event can be toggled on or off individually.

## Available Events

The following events are available for logging, organized by category:

### Moderation Events
Events related to server moderation and member discipline.

- **Member Banned**: Log when a member is banned from the server.
- **Member Unbanned**: Log when a member is unbanned from the server.
- **Member Kicked**: Log when a member is kicked from the server.
- **Member Warned**: Log when a member receives a warning.
- **Member Muted**: Log when a member is muted/timed out.

### Message Events
Events related to message activity within channels.

- **Message Deleted**: Log when a message is deleted.
- **Message Edited**: Log when a message is edited.
- **Bulk Messages Deleted**: Log when multiple messages are deleted at once.

### Member Events
Events related to members joining, leaving, or updating their profiles.

- **Member Joined**: Log when a new member joins the server.
- **Member Left**: Log when a member leaves the server.
- **Member Roles Updated**: Log when a member's roles are changed.
- **Member Nickname Changed**: Log when a member's nickname is changed.

### Channel Events
Events related to the creation, deletion, or modification of channels.

- **Channel Created**: Log when a new channel is created.
- **Channel Deleted**: Log when a channel is deleted.
- **Channel Updated**: Log when a channel is modified.

### Role Events
Events related to the creation, deletion, or modification of roles.

- **Role Created**: Log when a new role is created.
- **Role Deleted**: Log when a role is deleted.
- **Role Updated**: Log when a role is modified.

### Server Events
Events related to general server settings and assets.

- **Server Updated**: Log when server settings are changed.
- **Emoji Updated**: Log when emojis are created, deleted, or modified.
- **Invite Created**: Log when a new invite is created.
