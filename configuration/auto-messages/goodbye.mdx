# Goodbye Messages

MurFFy can also send a farewell message when a user leaves your server.

To get started, select **Goodbye** from the sidebar on the left in the dashboard.

## Enabling the Module

First, you need to enable the module to start configuring it.

![Enable Goodbye Module](https://cdn.murffy.xyz/docs/brave_zftHMdrEnK.png)

## Configuration

Once enabled, you can configure the following settings:

- **Goodbye Channel**: Select the channel where the goodbye message will be sent.
- **Message Type**: Choose between **Text** (plain text) or **Embed** (rich formatting).
- **Message Content**: Enter the content of your message or configure the embed fields.

### Variables

You can use the following variables in your message to make it dynamic:

| Variable | Description |
| :--- | :--- |
| `{mention}` | Mentions the user who left. |
| `{username}` | The username of the user. |
| `{server}` | The name of your server. |
| `{memberCount}` | The current member count of the server. |

## Random Messages

You can enable **Random Messages** to add multiple variations of your goodbye message. When a user leaves, the bot will randomly select one of the configured messages to send.
