# Goodbye Messages

MurFFy can also send a farewell message when a user leaves your server.

## Configuration

1. Go to the **Dashboard** and select your server.
2. Navigate to **Join Roles** -> **Auto Messages**.
3. Enable **Goodbye Messages**.
4. Customize your message content.

### Variables

| Variable | Description |
| :--- | :--- |
| `{user}` | The tag of the user who left. |
| `{username}` | The username of the user. |
| `{server}` | The name of your server. |
| `{count}` | The current member count of the server. |
