# Autoresponders

Autoresponders allow MurFFy to automatically reply to messages that contain specific keywords or phrases.

## Creating an Autoresponder

1. Go to the **Dashboard** and select your server.
2. Navigate to **Join Roles** -> **Auto Messages**.
3. Click on **Autoresponders**.
4. Click **Create New Autoresponder**.

### Settings

- **Trigger**: The phrase or keyword that triggers the response.
- **Response**: The message MurFFy will send back.
- **Match Type**: 
  - *Exact Match*: The message must be exactly the trigger phrase.
  - *Contains*: The message just needs to contain the trigger phrase.
  - *Wildcard*: Use wildcards for more complex matching.
