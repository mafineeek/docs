# Welcome Messages

MurFFy can automatically send a welcome message when a new user joins your server. This is a great way to greet new members, give them important information, or guide them to the rules channel.

To get started, select **Welcome** from the sidebar on the left in the dashboard.

## Enabling the Module

First, you need to enable the module to start configuring it.

![Enable Welcome Module](https://cdn.murffy.xyz/docs/brave_K5rzl9yHid.png)

## Configuration

Once enabled, you can configure the following settings:

- **Welcome Channel**: Select the channel where the welcome message will be sent.
- **Message Type**: Choose between **Text** (plain text) or **Embed** (rich formatting).
- **Message Content**: Enter the content of your message or configure the embed fields.

### Variables

You can use the following variables in your message to make it dynamic:

| Variable | Description |
| :--- | :--- |
| `{mention}` | Mentions the user who joined. |
| `{username}` | The username of the user. |
| `{server}` | The name of your server. |
| `{memberCount}` | The current member count of the server. |

## Random Messages

You can enable **Random Messages** to add multiple variations of your welcome message. When a user joins, the bot will randomly select one of the configured messages to send.

![Random Messages](https://cdn.murffy.xyz/docs/brave_UvWqnMNKIa.png)

## Direct Message

You can also enable **Direct Message** to send a private welcome message directly to the user's DM.

Like the server welcome message, you can choose between **Text** or **Embed** and customize the content.

![Direct Message Settings](https://cdn.murffy.xyz/docs/brave_ZefPq1ySeZ.png)
