# Suggestions

Enable a suggestions channel to let your members voice their ideas.

## Setup

1. **Enable the module** in the dashboard.
2. **Select a channel** where suggestions will be posted.

<img height="200" src="https://cdn.murffy.xyz/docs/brave_7rQd7jScks.png" />

## Features

### Auto-Create Threads

When enabled, a thread is automatically created for each new suggestion to allow for focused discussions.

<img height="200" src="https://cdn.murffy.xyz/docs/Discord_7hgW6rtsr3.png" />

### Use Messages

This setting allows members to simply send a message in the suggestions channel to create a suggestion, instead of using the `/suggest` command.

<img height="200" src="https://cdn.murffy.xyz/docs/Discord_cv89FdQbzD.gif" />
