# Tags

Create shortcuts for frequently used text or instructions.

## Creating a Tag

1. Select "Tags" from the sidebar and click **Create Tag**.

<img height="200" src="https://cdn.murffy.xyz/docs/brave_aKcThb9kN8.png" />

2. A modal window will open. Enter the **Trigger** (the name of the tag) and **Content** (the response message).

<img height="200" src="https://cdn.murffy.xyz/docs/brave_c4pNhvG5UB.png" />

## Managing Tags

After adding a tag, you can manage it from the list.

<img height="200" src="https://cdn.murffy.xyz/docs/brave_RCW1jaQ0fk.png" />

## Usage

To use a tag, type `/tag view` and search for the tag name.

<img height="200" src="https://cdn.murffy.xyz/docs/Discord_PhiGBineBz.png" />

**Result:**

<img height="200" src="https://cdn.murffy.xyz/docs/Discord_CjJQkyuElC.png" />

<Tip>You can also use `/tag list` to view all available tags directly on Discord.</Tip>
